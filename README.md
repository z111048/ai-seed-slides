## 核心痛點

都市設計審議報告書審查涵蓋章節完整性、書圖一致性、法規檢討等項目，規則明確但耗時。科室 7 名承辦人每年消耗約 **200 人日**於重複性格式比對，每次補正複查耗時約半天（4 小時）。

---

## AI 解決方案

採分階段、訂閱優先策略，五大技術模組：

- **P1（立即）**：LLM 格式檢核 + 補正意見草稿生成 + 會議摘要
- **P2（次階段）**：語音辨識逐字稿 + 版本差異追蹤
- **P3（後期）**：Vision LLM 書圖一致性影像辨識
